# Astro Static Output

Static-only example that mirrors an Astro `dist/` deploy. The checked-in
`dist/` directory is intentionally tiny; in a real Astro app it would be
generated by `astro build`.

Before deploying, edit `simple-vps.toml`:

- set `[env.production].server`
- set `[routes.site].host`

```bash
simple-vps check production
simple-vps setup production
simple-vps deploy production
curl https://site.example.com/
```
